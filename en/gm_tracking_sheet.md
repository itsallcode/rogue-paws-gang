# GM Tracking Sheet — Player Characters

This sheet is intended to be printed. Place tokens (e.g., pennies) on the squares to track luck and statuses. Clear the board between scenes as needed.

**Legend**

- Luck: move tokens when luck is lost or gained;
- Status boxes: place a token when the status is currently active. Remove when it ends.
- Uses: for limited-use abilities, fill one box per use spent; clear on rest as described.
